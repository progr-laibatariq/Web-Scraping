## üöÄ Features
* **Multi-Row Extraction:** Automatically loops through all products on a collection page.
* **Dynamic Selectors:** Uses specific CSS selectors (`span.money`) to accurately capture prices.
* **Clean Data Formatting:** Automatically cleans and exports data to `.xlsx` using the **openpyxl** engine.
* **Error Handling:** Gracefully manages missing elements to prevent script crashes.
